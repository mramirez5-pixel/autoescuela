# Autoescuela

## Requisitos
- XAMPP (Apache + MySQL)
- PHP 7+

## Instalación
1. Copiar carpeta `autoescuela` en `C:\xampp\htdocs\`
2. Abrir phpMyAdmin y ejecutar `sql/autoescuela.sql` para crear la base de datos y tablas
3. Abrir navegador:
   http://localhost/autoescuela/index.php

## Uso
- Clientes → CRUD
- Profesores → CRUD
- Vehículos → CRUD
- Clases → Crear clase automática y listar
